# India Runs — Data & AI Challenge
## Intelligent Candidate Ranking System

### What this does
Ranks 100,000 candidates for a Senior AI Engineer role
using a smart scoring system that thinks like a recruiter
— not just keyword matching.

### Scoring Logic
- Title relevance (30%) — filters irrelevant roles completely
- Career history (20%) — rewards product company experience
- Years of experience (20%) — sweet spot: 5-9 years
- Skills match (20%) — core AI/ML skills with trust weighting
- Behavioral signals (10%) — recency, response rate, notice period

### How to run
pip install pandas numpy
python rank.py

### Output
submission.csv — top 100 ranked candidates
